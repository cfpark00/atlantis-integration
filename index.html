Atlantis Integration Visualizations

Research Context

This project demonstrates a landmark finding in deep learning: neural networks can internally represent world properties underlying the data they're trained on, even when not explicitly instructed to do so.

We trained networks on synthetic tasks operating on city coordinates. By extracting and visualizing the learned representations, we can see how the network organizes spatial knowledge internally.

The following visualizations show what happens when we try to integrate new "Atlantis" cities into the model through fine-tuning. The key question: Does Atlantis integrate where it should in the representation space?

Understanding the Visualizations

PCA: Raw high-dimensional representations projected into 2D, revealing natural clustering patterns.

Not in Probe Training: Linear probes trained only on non-Atlantis cities. Shows where Atlantis naturally falls in the learned coordinate system.

In Probe Training: Linear probes trained on all cities including Atlantis to extract X and Y coordinates.
