Atlantis Integration Visualizations

Research Context

This project demonstrates a landmark finding in deep learning: neural networks can internally represent world properties underlying the data they're trained on, even when not explicitly instructed to do so.

We trained networks on synthetic tasks operating on city coordinates. By extracting and visualizing the learned representations, we can see how the network organizes spatial knowledge internally.

The following visualizations show what happens when we try to integrate new "Atlantis" cities into the model through fine-tuning. The key question: Does Atlantis integrate where it should in the representation space?

Good Generalization

When the fine-tuning task generalizes well, Atlantis cities are properly integrated into the representation space. They merge naturally with the existing city representations based on their actual spatial properties.

Understanding the Visualizations

PCA (Principal Component Analysis): Shows the raw high-dimensional representations projected into 2D, revealing natural clustering patterns.

In Probe Training: Linear probes are trained on all cities including Atlantis to find the dimensions that best extract X and Y coordinates.

Not in Probe Training: Linear probes are trained only on original cities, excluding Atlantis. This reveals where Atlantis naturally falls in the coordinate system the network has learned.
